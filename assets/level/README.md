# Level Assets

This folder contains static assets for the level system.

## awesome.png
Place your `awesome.png` image here. This image is used as the background for the leaderboard ranking cards.

The image is referenced in:
- `og/src/modules/level/index.ts` - in the `topBuilder()` function as the attachment for rank cards

**Recommended dimensions:** 1200x600px or wider (used for leaderboard background)
